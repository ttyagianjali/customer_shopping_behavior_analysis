# customer_shopping_behavior_analysis
This project analyzes customer purchasing behavior to identify revenue drivers, customer segments, and product performance patterns. Using Python and SQL, raw retail transaction data was cleaned, transformed, and analyzed to generate business insights that support marketing and customer retention decisions.
Customer Shopping Behavior Analysis

# Project Objectives
	•	Clean and preprocess raw customer transaction data
	•	Perform exploratory data analysis
	•	Segment customers based on purchase behavior
	•	Analyze subscription and discount impact on spending
	•	Identify top-performing products and revenue contributors

# Dataset

	•	Domain: Retail / E-commerce Analytics

# Key Features
	•	Customer demographics (Age, Gender)
	•	Product & category information
	•	Purchase amount
	•	Subscription status
	•	Shipping type
	•	Discounts applied
	•	Purchase history

# Data Cleaning & Preparation (Python):
Key preprocessing steps included:
	•	Handling missing review ratings using category median
	•	Standardizing column names
	•	Encoding Yes/No variables into binary values
	•	Creating Age Group segmentation
	•	Converting purchase frequency into numerical days
	•	Removing redundant columns

# SQL Business Analysis:
The following business questions were answered using SQL:
	1.	Revenue generated by gender
	2.	High-spending customers using discounts
	3.	Top-rated products
	4.	Average purchase comparison by shipping type
	5.	Spending behavior of subscribers vs non-subscribers
	6.	Products with highest discount usage
	7.	Customer segmentation (New, Returning, Loyal)
	8.	Most purchased products per category
	9.	Relationship between repeat buyers and subscriptions
	10.	Revenue contribution by age group

# Key Insights:
	•	Subscribers contribute higher overall revenue.
	•	Loyal customers represent a significant portion of sales.
	•	Discounts attract both value and high-spending customers.
	•	Certain products consistently achieve strong customer ratings.
	•	Customer demographics influence purchasing behavior.
